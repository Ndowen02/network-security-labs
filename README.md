# Network Security Labs

This repository contains hands-on labs completed as part of a Network Security course.  
The labs focus on practical experience with network analysis, reconnaissance, and security configuration.

## Environment
- Virtual machines used in isolated lab environments
- No production or public networks involved

## Tools & Technologies
- Kali Linux
- Wireshark
- Nmap
- Linux networking tools
- VirtualBox / VMware

## Lab Structure
Each lab folder includes:
- Lab objective
- Environment configuration
- Commands or configuration settings
- Results and security takeaways

## Disclaimer
All activities were performed in controlled environments for educational purposes only.
